# 🧪 UI Test Automation - Cypress  

## Page Objects + App Actions

Projeto de automação de testes E2E utilizando Cypress com aplicação dos padrões:

- Page Objects
- App Actions
- Factory Pattern
- Fixtures para massa fixa

---

## 🎯 Objetivo

Implementar testes automatizados organizados e escaláveis para fluxos de UI, separando:

- Mapeamento de elementos da interface (Page Objects)
- Fluxos de negócio reutilizáveis (App Actions)
